1. What are the main differences between external, internal, and inline CSS?

    External CSS:This is additional file that we create our styles. File name can be anything; however, it has to have '.css' extention at the end of the file. It has to be linked with the HTML file inside of the head section.

    Internal CSS: is used to define a style for a single HTML page. An internal CSS is defined in the head section of an HTML page, within a style element.

    Inline CSS:may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element.

  2. What is the syntax for class and ID selectors?

    For the class syntax it is used point '.' and for the id syntax it is used hashtag sign '#'

  3. How would you apply a single rule to two different selectors?

    We can write the two selectors next to each other with a coma between them like that: div, p{color: #000}

  4. Given an element that has an id of title and a class of primary, how would you use both attributes for a single rule?

    #title, .primary{}

  5. What does the descendant combinator do?

    Combines two selectors such that elements matched by the second selector are selected if they have an ancestor (parent) element matching the first selector. For example if we want to target p elemnt inside of a div element we can write like div p{}

  6. Between a rule that uses one class selector and a rule that uses three type selectors, which rule has the higher specificity?

    Class selector has the higher specificity.

  7. From inside to outside, what is the order of box-model properties?

    Content-padding-border-margin

  8. What does the box-sizing CSS property do?

    The box-sizing property defines how the width and height of an element are calculated: should they include padding and borders, or not.

  9. What is the difference between the standard and alternative box model?

    With the standart box model, the size of the border is added to the width and height of the box. With the alternative box model, the size of the border makes the content box smaller as it takes up some of that available width and height.

  10. Would you use margin or padding to create more space between 2 elements?

    For creating space between the two elements we can use margin.

  11. Would you use margin or padding to create more space between the contents of an element and its border?

    For creating space between the content of an element and its border we can use padding.

  12. Would you use margin or padding if you wanted two elements to overlap each other?

    We should use margin for overlapping elements.

  13. What is the difference between a block element and an inline element?

    A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. An inline element does not start on a new line and only takes up as much width as necessary.

  14. What is the difference between an inline element and an inline-block element?

    Compared to display: inline, the major difference is that inline-block allows to set a width and height on the element. Also, with display: inline, top and bottom margins & paddings are not respected, and with display: inline-block they are.

  15. Is an h1 block or inline?

    h1 is a block element.

  16. Is button block or inline?

    button is an inline level element.

  17. Is div block or inline?

    Div is a block-level element.

  18. Is span block or inline?

    Span is an inline level element.

  20. How do you create a flex item?

    We create an element inside of a div tag in the flex container.

  21. What are the 3 values defined in the shorthand flex property?

    Flex is a shorthand for flex-grow, flex-shrink and flex-basis.

  22. How do you make flex items arrange themselves vertically instead of horizontally?

    We set the flex-direction: column;

  23. What is the difference between justify-content and align-items?

    Justify-content is for horizantal aligning, align-items is for vertical aligning.

  24. How do you use flexbox to completely center a div inside a flex container?

    display: flex; justify-content: center; align-items: center;

  25. What’s the difference between justify-content: space-between and justify-content: space-around?

    Space-between: First item is on the start line and the last item is on the end line.

    Space around: Items are evenly distributed in the line with equal space around them.
